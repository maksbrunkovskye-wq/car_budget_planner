# Car Budget Planner
Ein einfacher Budget-Rechner in Python.

# Funktionen
- Man kann das Monatsgehalt eingeben.
- Man gibt die Kosten für Miete, Essen und andere Ausgaben ein.
- Das Programm berechnet den Restbetrag.
- Das Programm zeigt, ob das Budget gut oder kritisch ist.
- Es gibt ein einfaches Menü im Terminal.

# Technologien
- Python 3
- Funktionen
- Schleifen
- Bedingungen (if / elif / else)
